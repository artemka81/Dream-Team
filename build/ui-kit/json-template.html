Данные из JSON файла

Всем кто хочет попрактиковаться в работе с данными из json, я подготовил шаблон этой странице. В проекте я добавил json-файл с направлениями авиаперелетов перелетов (app/date/menu.json).

Вам нужно:

  1. Подключение

    В сборку этого проекта уже подключен плагин "gulp-data" который подключает файл с данными и плагин fs который преоразовывает json в массив. В файле конфигурации проекта gulpfile.js уже подключен json файл в таксе "pug".

  2. Выводим данные в шаблон

    Данные из файла находяться в глобальной области видимости, поэтому вы можете обращаться к данным из любого файла в проекте. Например, что бы получить количество авианаправлений из файла, можно так #{menu.fly.length}

    Кол-во направлений — 5

    Первое направление в списке #{menu.fly[0].flyIn} - #{menu.fly[0].flyOut} - Цена билета #{menu.fly[0].price}

    Москва - Калуга - Цена билета 1734

  3. Выводим данные в шаблон
